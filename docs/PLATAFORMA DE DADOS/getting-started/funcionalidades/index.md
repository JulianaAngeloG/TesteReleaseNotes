A Data Platform oferece um conjunto completo de ferramentas para ingestão, transformação, visualização e gerenciamento de dados.

Com funcionalidades intuitivas, ela facilita o trabalho de profissionais de dados, analistas e equipes que buscam uma experiência eficiente e segura na exploração de dados. A seguir, as principais funcionalidades da plataforma.

## Ingestão de Dados

A ingestão de dados é um processo essencial para trazer informações de diferentes fontes para a plataforma, garantindo que os dados possam ser analisados de forma organizada e acessível.

* **Ingestão Completa (carga full)**: A carga de dados full transfere todos os registros de uma ou mais tabelas do sistema de origem para a plataforma. Isso significa que todas as vezes que o processo de ingestão for executado, os dados na plataforma são excluídos e uma nova carga é completa é realizada.
* **Ingestão Incremental (carga incremental)**: atualize apenas os dados novos ou alterados desde a última ingestão, otimizando o processo e economizando recursos.
* **Upload de Planilhas:** envie arquivos em formato CSV para adicionar novos dados à plataforma de forma rápida e prática.

## Catálogo de Dados

O catálogo de dados organiza e facilita a busca pelas informações armazenadas na plataforma.

Acesse o catálogo completo de dados, que lista todas as tabelas e fontes de dados disponíveis na plataforma, tornando mais fácil localizar as informações necessárias para suas análises.

## Transformação de Dados

Com a ferramenta de transformação, é possível combinar, limpar e organizar os dados de maneira personalizada para atender às necessidades da sua análise.

Combine dados de duas ou mais tabelas para criar conjuntos de dados mais completos, adequados a diferentes necessidades de análise e visualização.

## Visualização de Dados

As funcionalidades de visualização permitem explorar os dados de forma gráfica, facilitando a compreensão e análise dos mesmos. A ferramenta de visualização de dados da plataforma é a Sankhya Analytics AI.

Dashboard: acompanhe estatísticas e métricas dos dados em tempo real, com gráficos e indicadores personalizáveis que ajudam a monitorar o desempenho da sua operação.

## Logs

Acompanhe o funcionamento da plataforma e fique atento a possíveis problemas ou falhas.Tenha acesso à lista de logs, registrando problemas e falhas que possam ter ocorrido durante o uso da plataforma, permitindo rastrear e solucionar quaisquer incidentes.
