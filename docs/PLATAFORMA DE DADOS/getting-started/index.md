# O que é a Data Platform Sankhya?

A Data Platform Sankhya oferece uma solução integrada para gestão, transformação, visualização e Governança de Dados.  Ela conta com ferramentas que permitem o acesso rápido a informações cruciais, otimizando o processo de tomada de decisão e promovendo a eficiência no uso de dados dentro da organização.
