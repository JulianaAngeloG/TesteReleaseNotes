Para acessar a plataforma de dados, entre em contato com [joao.piotto@sankhya.com.br](mailto:joao.piotto@sankhya.com.br) e solicite a integração do seu usuário SankhyaID com a plataforma. Você receberá um email com as credenciais do Athena para consulta de dados e instruções para acessar a plataforma.

📌 [Acesse aqui a Plataforma de Dados](http://sankhya-data-platform-ui.s3-website-sa-east-1.amazonaws.com/).
