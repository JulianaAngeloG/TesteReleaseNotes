## Política de acesso

Por motivos de segurança, todos os dados presentes na Data Platform contam com mecanismos de Políticas de Acesso. Isso significa que cada usuário consegue visualizar apenas as informações pertinentes ao seu perfil.

A Política de Acesso é fundamental para garantir a Governança de Dados e a segurança de dados sensíveis.Saiba mais acessando a seção Governança de Dados.
